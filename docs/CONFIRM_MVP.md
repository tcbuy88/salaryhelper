# 确认 MVP（必须上线的功能）

版本：v1

目标：最小化可交付、闭环的核心功能，支持用户从咨询到付费与文书生成。

必须上线（按优先级）
1. 用户认证
   - 手机号 + 验证码登录
   - JWT + refresh token

2. AI 聊天
   - 会话创建/列表/消息发送
   - 模型调用（异步处理），AI 回写消息
   - 会话历史查看与导出（txt/pdf）
   - 免费额度与会员模型限制

3. 文书生成（DocGen）
   - 模板管理（后端）
   - 前端基于 template.fields 动态表单渲染
   - 渲染/预览/导出并保存 file_id 关联会话

4. 支付与订单
   - SKU 管理、订单创建
   - 微信 H5 / 支付宝 H5 支付接入（含回调验签）
   - 权益发放（会员/次数）

5. 文件/证据管理
   - 文件上传（对象存储）、证据列表

6. 管理后台（基础）
   - 用户/订单/会话/模板 浏览与导出
   - 人工复核入口

非 MVP（可后期）
- 语音输入、律师对接、完整分账、开票自动化
